# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

Jika Anda menemukan kerentanan keamanan dalam proyek ini, mohon hubungi kami dengan cara:

1. **Jangan** buat issue publik untuk kerentanan
2. Kirim detail kerentanan ke email security@example.com
3. Berikan detail teknis dan proof of concept
4. Kami akan merespons dalam 48 jam

## Security Best Practices

Proyek ini:
- Hanya membaca direktori, tidak mengubah file
- Tidak mengirim data ke eksternal
- Berjalan dengan permission user saat ini
- Tidak menyimpan data sensitif

## Known Issues

Tidak ada known security issues saat ini.

## Updates

Security updates akan diumumkan melalui:
- GitHub Security Advisories
- CHANGELOG.md
- Release notes
